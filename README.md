# 🏏 Cricket Legends - React Application

An **extraordinary** React.js web application showcasing 15 legendary cricket players with stunning animations, interactive features, and modern design.

## ✨ Features

### 🎨 **Stunning Visual Design**
- **Gradient Backgrounds**: Dynamic mesh gradients with animated orbs
- **Particle Effects**: Floating cricket balls and twinkling stars
- **Glass Morphism**: Modern frosted glass effects with backdrop blur
- **3D Effects**: Depth and dimension with shadows and glows
- **Responsive Design**: Perfectly adapts to all screen sizes

### ⚡ **Advanced Animations**
- **Framer Motion**: Smooth page transitions and micro-interactions
- **Loading Screen**: Engaging animated loader with progress bar
- **Card Animations**: Hover effects, scale transforms, and smooth transitions
- **Staggered Animations**: Sequential element appearances
- **Parallax Effects**: Dynamic header movement on scroll

### 🎯 **Interactive Features**
- **Role Filtering**: Filter players by Batsmen, Bowlers, or All-rounders
- **Search Functionality**: Real-time search by name, country, or role
- **View Modes**: Toggle between Grid and List layouts
- **Modal Details**: Click any player for detailed statistics and biography
- **Theme Toggle**: Switch between dark and light modes
- **Live Stats Panel**: Dynamic statistics with animated counters

## 🚀 **Getting Started**

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

The app will open automatically at [http://localhost:3000](http://localhost:3000)

## 🎮 **How to Use**

1. **Browse Players**: Scroll through the grid of legendary players
2. **Filter**: Click role buttons to filter by player type
3. **Search**: Type in the search bar to find specific players
4. **View Details**: Click any card to see detailed stats and bio
5. **Toggle View**: Switch between grid and list layouts
6. **Change Theme**: Click the sun/moon icon to toggle themes
7. **Explore Stats**: Check the statistics panel for quick insights
